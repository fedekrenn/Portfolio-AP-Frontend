Cocinando Fácil | Web para gestión de recetas y obtener ideas

Proyecto final de cursada Javascript en Coderhouse. El mismo es un simulador de cocina que permite cargar recetas, compararlas, filtrar por comidas rápidas y aquellas que son aptas para veganos. Además cuenta con conexión a una API que permite ingresar un ingrediente y la web automáticamente te devuelve recetas que podés cocinar con dicho ingrediente. Como añadido tiene función modo oscuro. Las recetas guardadas son almacenadas en el localStorage para tenerlas disponibles en una próxima sesión de navegación. Tecnologías utilizadas: Html, Css, Javascript.

Sereno Bienestar | Un lugar para encontrarte contigo mismo

Proyecto final de cursada de Desarrollo Web en Coderhouse. Es una web de 5 páginas maquetada con flexbox en mayor medida y grid. Además se usa framework de bootstrap e incluye el pre-procesador de Sass. Cuenta con librería de animaciones y está optimizada para SEO y a la vez es responsive, se puede visualizar cómodamente el contenido tanto en mobile, tablets o desktop.
